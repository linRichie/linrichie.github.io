Python 的 os 模块提供了访问和操作操作系统的功能。它包含了许多函数和变量，可以用于处理文件、目录、环境变量等。

# 导入 os 模块

要使用 os 模块，首先需要导入它：

import os

# 获取当前工作目录

os.getcwd() 函数可以获取当前工作目录：

>>> os.getcwd()
'/home/user'

# 设置当前工作目录

os.chdir() 函数可以设置当前工作目录：

>>> os.chdir('/tmp')
>>> os.getcwd()
'/tmp'

# 创建目录

os.mkdir() 函数可以创建目录：

>>> os.mkdir('my_dir')

# 删除目录

os.rmdir() 函数可以删除空目录：

>>> os.rmdir('my_dir')

# 列出目录中的文件和目录

os.listdir() 函数可以列出目录中的文件和目录：

>>> os.listdir('.')
['.bashrc', '.profile', '.git', 'README.md']

# 获取文件大小

os.path.getsize() 函数可以获取文件的大小：

>>> os.path.getsize('README.md')
252

# 判断文件是否存在

os.path.exists() 函数可以判断文件是否存在：

>>> os.path.exists('README.md')
True

# 创建文件

open() 函数可以创建文件：

>>> with open('new_file.txt', 'w') as f:
...     f.write('Hello, world!')
...

# 读取文件

open() 函数也可以用于读取文件：

>>> with open('new_file.txt', 'r') as f:
...     contents = f.read()
...
>>> contents
'Hello, world!'

# 删除文件

os.remove() 函数可以删除文件：

>>> os.remove('new_file.txt')

# 修改文件权限

os.chmod() 函数可以修改文件权限：

>>> os.chmod('README.md', 0o777)

# 获取文件所有者

os.stat() 函数可以获取文件的所有信息，包括所有者：

>>> stat = os.stat('README.md')
>>> stat.st_uid
1000

# 获取文件组

os.stat() 函数还可以获取文件组：

>>> stat = os.stat('README.md')
>>> stat.st_gid
1000

# 获取环境变量

os.environ 变量是一个字典，包含了所有环境变量：

>>> os.environ['HOME']
'/home/user'

# 设置环境变量

os.environ 变量也可以用于设置环境变量：

>>> os.environ['PATH'] = '/usr/bin:/bin'

# 获取操作系统类型

os.name 变量可以获取操作系统类型：

>>> os.name
'posix'

# 获取操作系统版本

sys.version 变量可以获取操作系统版本：

>>> sys.version
'3.12.1 (default, Oct 19 2023, 14:32:09) \n[GCC 12.1.0]'

# 以下是一些 os 模块的其他常用功能：

# os.system()

  • os.system() 函数可以运行系统命令：
>>> os.system('ls -al')

输出：

total 16
drwxr-xr-x 1 user 1000 4096 Jan  3 15:23 .
drwxr-xr-x 1 user 1000 4096 Jan  3 15:23 ..
-rw-r--r-- 1 user 1000  252 Jan  3 15:23 README.md

# os.path.join()

  • os.path.join() 函数可以将多个路径组合成一个路径：
>>> os.path.join('/home/user', 'data', 'file1.txt')
'/home/user/data/file1.txt'

# os.path.split()

  • os.path.split() 函数可以将路径拆分为目录和文件名：
>>> os.path.split('/home/user/data/file1.txt')
('/home/user/data', 'file1.txt')

# os.path.isfile()

  • os.path.isfile() 函数可以判断路径是否指向文件：
>>> os.path.isfile('/home/user/data/file1.txt')
True

# os.path.isdir()

  • os.path.isdir() 函数可以判断路径是否指向目录：
>>> os.path.isdir('/home/user/data/file1.txt')
False

# os.path.getmtime()

  • os.path.getmtime() 函数可以获取文件的最后修改时间：
>>> os.path.getmtime('/home/user/data/file1.txt')
1641296309.0

# os.path.getsize()

  • os.path.getsize() 函数可以获取文件的大小：
>>> os.path.getsize('/home/user/data/file1.txt')
252

# os.path.realpath()

  • os.path.realpath() 函数可以获取文件的绝对路径：
>>> os.path.realpath('/home/user/data/file1.txt')
'/home/user/data/file1.txt'

# os.path.relpath()

  • os.path.relpath() 函数可以计算两个路径之间的相对路径：
>>> os.path.relpath('/home/user/data/file1.txt', '/home/user/data')
'file1.txt'

# os.path.walk()

  • os.path.walk() 函数可以遍历一个目录树中的所有文件和目录：
for dirpath, dirnames, filenames in os.walk('/home/user'):
    for filename in filenames:
        print(os.path.join(dirpath, filename))

输出：

/home/user/.bashrc
/home/user/.profile
/home/user/data/file1.txt
/home/user/data/file2.txt

# os.path.relpath()

  • os.path.relpath() 函数用于计算两个路径之间的相对路径
import os.path

path1 = "/home/user/data/file1.txt"
path2 = "/home/user/data"

rel_path = os.path.relpath(path1, path2)

print(rel_path)

输出：

file1.txt

在上述示例中，path1 是绝对路径，path2 是相对路径。os.path.relpath() 函数将 path1 相对于 path2 的路径计算为 file1.txt。

os.path.relpath() 函数的参数如下：

  • path: 要计算相对路径的路径
  • start: 相对路径的起始路径

如果 start 为 None，则 os.path.relpath() 函数将 path 视为相对于当前工作目录的路径。

# os.path.join()

os.path.join() 函数用于将多个路径组合成一个路径。

import os.path

path1 = "/home/user"
path2 = "data/file1.txt"

combined_path = os.path.join(path1, path2)

print(combined_path)

输出：

/home/user/data/file1.txt

在上述示例中，os.path.join() 函数将 path1 和 path2 组合成 /home/user/data/file1.txt 路径。

os.path.join() 函数的参数如下：

  • paths: 要组合成路径的路径列表

# os.walk()

os.walk() 函数用于遍历一个目录树中的所有文件和目录。

import os.path

for dirpath, dirnames, filenames in os.walk("/home/user"):
    for filename in filenames:
        print(os.path.join(dirpath, filename))

输出：

/home/user/.bashrc
/home/user/.profile
/home/user/data/file1.txt
/home/user/data/file2.txt

在上述示例中，os.walk() 函数将 /home/user 目录树中的所有文件和目录遍历一遍。对于每个目录，os.walk() 函数将返回三个值：

  • dirpath: 目录的路径
  • dirnames: 目录下的子目录列表
  • filenames: 目录下的文件列表

可以使用这些值来处理目录树中的文件和目录。

总结

os.path.relpath()、os.path.join() 和 os.walk() 是三个常用的 os.path 模块函数。它们可以用于处理路径、遍历目录树等任务。

# 总结

os 模块提供了许多用于操作操作系统的功能。通过使用 os 模块，可以轻松地访问和操作文件、目录、环境变量等。
